EasyTrip - Local Train ticket booking website
hosted @ https://localtat.000webhostapp.com/

Till now there is no site that book local train ticket. Therefore, there is haphazard seating arrangement. We are going to manage the crowd and will establish regular seating arrangement. The cash at the end of the day have to be kept safe. Since the existing system is also paper based it has a big disadvantage of data integrity, calculations have to be done manually and the data can be easily manipulated or lost. In our proposed, the issue of not only payments but the entire hectic management of a train ticket booking is easily managed. There is no need for long queues, Tickets can be booked online. No paper-based records are need to be maintained. Migration of data to the cloud is needed only once. It automates the entire train ticket booking system.
